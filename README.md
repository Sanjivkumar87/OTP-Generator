# OTP-Generator
🔐 OTP Generator – Hacker Style UI

A stylish and secure OTP Generator web application built using HTML, CSS, JavaScript, and Bootstrap. This project generates random OTPs with interactive message alerts, sound effects, and a futuristic hacker-themed UI/UX interface for an engaging user experience.

✨ Features
🔢 Generate Random OTP instantly
🔊 OTP generation sound effect
💬 Animated success/message alerts
💻 Hacker-style dark UI design
⚡ Responsive and modern interface
🎨 Built with Bootstrap for clean layout
📱 Mobile-friendly design
🚀 Fast and lightweight project
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Bootstrap 5
📌 Project Highlights

This project combines frontend development with creative UI/UX design. The hacker-themed interface, glowing effects, and sound notifications make the OTP generation process more interactive and visually appealing.
